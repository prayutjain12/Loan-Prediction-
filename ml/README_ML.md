# Machine Learning for Loan Prediction

This directory contains the machine learning core for the Loan Prediction System. While the website provides a high-fidelity UI simulation, these scripts represent the actual "brain" behind the predictions.

## Files
- `generate_dataset.py`: Generates a synthetic dataset of 1,000 loan applicants with various features (Income, Credit History, etc.).
- `train_model.py`: Preprocesses the data and trains a **Random Forest Classifier** to predict loan eligibility.
- `loan_data.csv`: The dataset used for training (generated by `generate_dataset.py`).
- `loan_prediction_model.pkl`: The trained model file (generated by `train_model.py`).
- `requirements.txt`: Python dependencies.

## How to Run
1. Install dependencies:
   ```bash
   pip install -r ml/requirements.txt
   ```
2. Generate the dataset:
   ```bash
   python ml/generate_dataset.py
   ```
3. Train the model:
   ```bash
   python ml/train_model.py
   ```

## Integration
The website (`src/pages/Eligibility.jsx`) currently uses heuristic logic derived from this model's rules. For production use, this model can be served via a FastAPI or Flask backend and called by the React frontend via Fetch API.
